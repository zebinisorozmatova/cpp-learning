# C++ Learning Repository

Welcome to my C++ learning journey!  
This repository contains beginner-friendly C++ programs designed to help learners understand the basics of C++ step by step.

---

## Projects Overview

Each program is explained clearly with comments and step-by-step guidance to make learning easier:

- **hello.cpp** – Prints "Hello GitHub" and introduces basic output in C++.
- **odd_even.cpp** – Checks if a number is odd or even using if-else statements.
- **calculator.cpp** – A simple program to practice arithmetic operations.
